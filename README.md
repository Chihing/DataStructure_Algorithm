 ### 学习思路建议：

 - 先理解算法概念，了解其原理，知道其流程
 
 - 开始敲代码，第一、二遍照着本项目源码边敲边理解
 
 - 再后来，不要看源码，自己实现算法，哪里卡住就从头梳理思路，尝试自己解决

 - 最后还要手写这些算法，大公司面试一般都会让在纸上写伪代码，提前练习以防止到时候手生。

 个人觉得重要的是做练习。算法的话练习得越多，对算法就越了解，也就越能够深刻记住这种算法。因为记忆前有理解作为前提的话，效果会非常好的。
